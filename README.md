# Movie-viewer

_Proyecto de Visualización de peliculas usando la API de [TMDb](https://www.themoviedb.org/)_

## Comenzando 🚀

_Este proyecto contiene lo siguiente:

  - [Kotlin Coroutines]
  - [MVVM]
  - [Clean Architecture]
  - Modularization per layer (Basic).
  - Unit Test
_
